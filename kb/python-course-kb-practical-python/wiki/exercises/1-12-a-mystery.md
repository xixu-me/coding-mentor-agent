# Exercise 1.12: A Mystery

> Source: Practical Python Programming, `01_Introduction/03_Numbers.md`.

### Exercise 1.12: A Mystery

`int()` and `float()` can be used to convert numbers.  For example,

```python
>>> int("123")
123
>>> float("1.23")
1.23
>>>
```

With that in mind, can you explain this behavior?

```python
>>> bool("False")
True
>>>
```
